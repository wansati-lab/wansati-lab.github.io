# Contributing to Wansatilab Website

Thank you for your interest in contributing to the Wansatilab website!

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test locally with `bundle exec jekyll serve`
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature`
7. Submit a pull request

## Guidelines

- Follow Jekyll best practices
- Ensure the site builds without errors
- Test changes in multiple browsers
- Keep the design consistent with the existing theme
