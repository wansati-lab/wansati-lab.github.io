# Wansatilab Website

This is the official website for Wansatilab, a Mozambican association focused on women in technology.

## About

Wansatilab is dedicated to empowering women and girls in Mozambique through technology education, entrepreneurship, and community building.

## Development

This site is built with Jekyll and hosted on GitHub Pages.

### Prerequisites

- Ruby (version 2.7 or higher)
- Bundler

### Setup

1. Clone the repository
2. Install dependencies: `bundle install`
3. Run the site locally: `bundle exec jekyll serve`
4. Open http://localhost:4000

### Structure

- `_includes/`: Reusable HTML components
- `_layouts/`: Page layouts
- `_pages/`: Additional pages
- `assets/`: CSS, JS, images, and other assets
- `docs/`: Documentation
